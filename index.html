Halftone

Compose layered halftone patterns using geometric shapes.

Field

The background pattern covering the whole canvas. Use Size to control dot diameter or line thickness, and Gap to control spacing between them. Switch between dots, lines, or blank.

Adding shapes

Click the circle, square, or triangle buttons in the panel to add a shape to the canvas.

Moving

Drag the interior of any shape to reposition it.

Resizing

Drag near the edge of a shape to resize it. On a square, drag the left or right edge to change width, or the top or bottom edge to change height — making it a rectangle.

Rotating

Hover over a shape to reveal the circular handle near its top edge. Drag the handle to rotate the shape freely.

Density & pattern

Right-click a shape to open its property box. Use Size to control dot diameter or line thickness, Gap to set spacing, and the pattern buttons to switch between dots and lines. Toggle ■ for a solid background and ◑ to invert colours. Use Angle to rotate precisely, and W / H to resize.

Freeform path

Click the pen icon to start drawing. Each click places a corner point; click and drag to create a smooth curve point (handles appear). Click the first point again, or press Enter, to close the shape. Press Escape to cancel. After closing, double-click the shape to enter edit mode — drag anchor squares to move points, drag handle circles to adjust curves. Press Escape to exit edit mode.

Keyboard shortcuts

Delete / Backspace — delete selected shape
Arrow keys — nudge 1 px; hold Shift for 10 px

Layers

Each shape is listed under Layers. The top entry is the frontmost shape. Drag entries to reorder. Click ● to toggle visibility, × to delete.

Saving & loading

PNG — exports a raster image; white areas are saved as transparent.
SVG — exports a scalable vector file with transparent background.
JSON — saves the full project (all shapes, settings, and layers) as a file that can be reloaded.
Load JSON — opens a previously saved project file, replacing the current canvas.
